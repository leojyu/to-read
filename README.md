# To See-Books

> 一个极简、优雅的纯前端电子书阅读与出版平台，所有数据存储在您的浏览器本地。

**To See-Books** 是一个基于浏览器的个人图书管理工具，支持您注册登录、上传 `.txt` 格式的小说，并随时阅读、管理自己的作品。它无需服务器，完全运行在客户端，您的书稿和阅读记录都安全地保存在您的设备上。

---

## ✨ 功能特性

- **用户系统**：注册 / 登录，每个用户拥有独立的书库和阅读记录
- **书籍管理**：
  - 上传 `.txt` 文件作为小说正文，自动提取内容并生成预览
  - 填写书名、作者、简介，一键「出版」
  - 支持删除自己出版的作品
- **阅读体验**：
  - 点击任意书籍即可进入沉浸式阅读模式
  - 支持全屏阅读（自动触发浏览器全屏）
  - 自动记录最近阅读的 20 本书
- **数据持久化**：
  - 所有数据（用户、书籍、阅读记录）保存在 `localStorage`
  - 支持导出为 JSON 文件，方便备份或迁移
  - 支持导入之前导出的 JSON 数据（覆盖式恢复）
- **草稿自动保存**：在「出版」表单中输入的内容会自动保存为草稿，关闭页面再打开依然存在（仅登录用户有效）
- **轻量级 UI**：无任何外部依赖，响应式设计，适配桌面与移动端

---

## 🛠️ 技术栈

- 原生 HTML5 + CSS3 + JavaScript（ES6）
- `localStorage` 作为数据存储引擎
- 无第三方库，零依赖

## 🚀 如何使用

### 直接运行

下载 `To See-Books.html` 文件，用浏览器（推荐 Chrome / Edge / Firefox）打开即可使用。无需任何服务器或网络连接。

### 初次使用

1. **注册账号**：点击「出版」标签，在登录框下方选择「注册」，填写用户名和密码即可。
2. **登录**：注册后自动登录，或使用已有账号登录。
3. **出版书籍**：登录后，在「出版」标签下填写书名、作者、简介，并上传一个 `.txt` 文件（UTF-8 编码），点击「出版」即可将书籍加入书库。
4. **阅读**：在「最近看过」或「发现」列表中点击任意书籍，进入阅读模式。按 `ESC` 键或点击「关闭」按钮退出阅读。
5. **管理**：只有您自己出版的书才会显示「删除」按钮，可随时移除。
6. **数据备份**：使用页脚「导出数据」按钮保存所有书籍和阅读记录为 JSON 文件；使用「导入数据」可恢复之前备份的数据（**注意：导入会覆盖当前全部数据**）。

---

## 📁 数据存储说明

- 所有数据存储在浏览器的 `localStorage` 中，不同浏览器或设备之间**不互通**。
- 清除浏览器缓存或网站数据将导致所有数据丢失，请务必定期导出备份。
- 用户密码以明文方式存储（因为仅作为本地身份标识，无安全风险，但仍不建议使用重要密码）。

## 🧪 开发与扩展

### 本地开发
本项目为单 HTML 文件，无需构建工具。您可以直接修改文件，在浏览器中刷新查看效果。

### 数据格式
- `toseebooks_users`：`{ username: password, ... }`
- `toseebooks_books`：`[ { id, title, author, desc, content, owner, publishedAt }, ... ]`
- `toseebooks_recent`：`[ bookId, ... ]`
- `toseebooks_draft`：`{ title, author, desc, fileName, content }`（登录用户草稿）
